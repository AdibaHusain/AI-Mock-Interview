# MockMind AI  

AI-powered mock interview platform with voice interaction, real-time evaluation, and performance reports.


**Live Demo** → https://mockmind-ai.vercel.app/

## Overview
MockMind AI simulates real job interviews end-to-end. Users paste a job description, receive 6 tailored interview questions generated by LLaMA 3.3, answer each question verbally using their microphone, and receive instant AI feedback — including a score, written evaluation, and a model answer for every response. A comprehensive performance report is generated at the end of each session.

## Features

Role-specific question generation — 2 technical, 2 behavioural, 1 system design, 1 situational questions parsed from the job description
Voice-first interview experience — questions are read aloud via Text-to-Speech; answers are captured via Speech-to-Text in real-time
Per-answer AI evaluation — score out of 10, written feedback, and a model answer after each response
Performance report — strengths, weaknesses, improvement tips, and an overall summary after session completion
Session history — all past interviews tracked with scores and status on the dashboard
Secure authentication — JWT-based auth with bcrypt password hashing
Aurora animated background — Three.js WebGL shader background

## How the AI Works
***Question Generation***
The job role and description are sent to Groq's LLaMA 3.3 70B model with a structured prompt requesting exactly 6 questions in a defined format across four categories.
***Answer Evaluation***
Each answer is evaluated against the question. The model returns a score (0–10), written feedback identifying what was strong or missing, and a refined model answer showing what a comprehensive response would look like.
***Report Generation***
After all answers are submitted, the full Q&A set with scores is analysed to produce an overall performance report covering strengths, improvement areas, actionable tips, and a summary.
